# Hexadecimal Audio Transmission

Project developed for the Digital Systems and Computer Architecture course.

## Description
The program converts a student's academic registration number (RA) into a hexadecimal sequence and transmits it as audio frequencies stored in a WAV file. Each hexadecimal digit is mapped to a musical frequency and later decoded using Fast Fourier Transform (FFT).

## Technologies
- Python
- NumPy
- SciPy
- FFT

## Files
- lab1.py → main program
- 23005304.wav → generated audio signal
